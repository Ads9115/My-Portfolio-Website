A retro, interactive browser-based operating system built to showcase my graphics programming and game development portfolio. Built entirely with vanilla HTML, CSS, and JavaScript.

## Features
* Interactive Desktop: Draggable windows, clickable icons, and a working Start Menu.
* Custom Mini-Apps: A functional command prompt, a software 3D rasterizer, and a pixel paint tool.
* Living Environment: Animated background, wandering desktop pet, and physics-based interactions.

## Usage
Simply open `index.html` in any modern web browser. No local server or build tools required.
